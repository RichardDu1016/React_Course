# React Tic Tac Toe

這是一個使用 React 開發的井字棋（Tic Tac Toe）小遊戲，作為 React 練習課程的一部分。專案練習了 component 組合、狀態管理（useState）以及事件處理。

## 🚀 功能介紹

- 玩家輪流下棋（X / O）
- 自動判斷勝負
- 點擊棋盤可落子
- 遊戲結束後可重新開始

## 🛠 使用技術

- React (useState)
- JavaScript (ES6)
- CSS Modules / 自訂樣式
- Vite

## ▶️ 使用方式

```bash
git clone https://github.com/你的帳號/react-course-practice.git
cd tictactoe
npm install
npm run dev  # 或 npm start
```
